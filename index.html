What is B Zero X?

B Zero X is a revolutionary DeFi protocol that combines proof-of-work mining with Uniswap v4 liquidity provision, creating a sustainable ecosystem for yield generation through mining rewards and liquidity staking.

Protocol Overview

B Zero X represents the next evolution in decentralized finance, integrating proof-of-work consensus mechanisms with advanced liquidity provisioning strategies. Built on top of Uniswap v4's hook system, our protocol enables users to earn rewards through multiple mechanisms while maintaining decentralization and security.

Key Features

  • Proof-of-Work Mining: Mine B0x tokens using computational power, ensuring fair distribution
  • Uniswap v4 Integration: Advanced hooks for optimized liquidity provisioning
  • Token Bridging: Seamless conversion between B0x and 0xBitcoin
  • Staking Rewards: Earn continuous rewards through liquidity provision
  • Permissionless Access: No gatekeepers, fully decentralized protocol

Architecture

The B Zero X protocol consists of several interconnected smart contracts that work together to provide a comprehensive DeFi experience. The core components include mining contracts, liquidity management, and reward distribution systems.

Mining Algorithm

B Zero X uses a modified proof-of-work algorithm optimized for fair distribution and energy efficiency. The mining difficulty adjusts automatically to maintain consistent block times.

Technical Specifications

  • Algorithm: Keccak256-based hash function
  • Block Time: ~10 minutes average
  • Difficulty Adjustment: Every 1024 blocks
  • Max Supply: 21,000,000 B0x tokens
  • Halving: Every 2,100,000 blocks

Mining Process

Miners compete to solve cryptographic puzzles by finding nonces that produce hash values meeting the current difficulty target. Successful miners receive newly minted B0x tokens as rewards.

How to Mine B0x Tokens

Step-by-step guide to start mining B0x tokens using proof-of-work consensus.

Getting Started

Prerequisites

  • Ethereum wallet (MetaMask, WalletConnect, etc.)
  • ETH for gas fees
  • Mining software (provided below)
  • Computer with GPU (recommended) or CPU

Mining Setup

1. Download Mining Software:

git clone https://github.com/bzero-x/b0x-miner cd b0x-miner npm install

2. Configure Your Wallet:

export PRIVATE_KEY="your_private_key_here" export RPC_URL="https://mainnet.infura.io/v3/your_project_id"

3. Start Mining:

npm run mine

Mining Tips

  • Monitor gas prices for optimal submission timing
  • Use GPU mining for better hash rates
  • Join mining pools for consistent rewards
  • Keep your mining software updated

Hook Address

Understanding hook addresses and their role in the B Zero X protocol.

Hook Address System

Hook addresses in Uniswap v4 determine which hooks are called during pool operations. B Zero X uses specific hook addresses to enable enhanced functionality.

Hook Address Format

Hook addresses encode hook permissions in their bits:

  • Bit 0: beforeInitialize
  • Bit 1: afterInitialize
  • Bit 2: beforeAddLiquidity
  • Bit 3: afterAddLiquidity
  • Bit 4: beforeRemoveLiquidity
  • Bit 5: afterRemoveLiquidity
  • Bit 6: beforeSwap
  • Bit 7: afterSwap

B Zero X Hook Address

Main Hook: 0x8000000000000000000000000000000000000001

This address enables beforeSwap and afterSwap hooks for reward distribution.

B0x ↔ 0xBitcoin Conversion

Technical details of the conversion mechanism between B0x and 0xBitcoin tokens.

Conversion Architecture

The conversion system enables seamless exchange between B0x and 0xBitcoin ecosystems, maintaining a 1:1 peg while preserving the unique properties of each token.

Technical Implementation

contract BZeroXConverter { function convertToOxBitcoin(uint256 amount) external { require(bzeroX.transferFrom(msg.sender, address(this), amount)); oxBitcoin.transfer(msg.sender, amount); emit Converted(msg.sender, amount, true); } function convertToBZeroX(uint256 amount) external { require(oxBitcoin.transferFrom(msg.sender, address(this), amount)); bzeroX.transfer(msg.sender, amount); emit Converted(msg.sender, amount, false); } }

Conversion Features

  • 1:1 Ratio: Fixed exchange rate
  • Instant Conversion: No waiting periods
  • Gas Optimized: Minimal transaction costs
  • Reversible: Convert back anytime
